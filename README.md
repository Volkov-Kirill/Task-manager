# Task Manager WPF

## 📌 Описание проекта

Приложение для управления задачами (Task Manager), разработанное на C# с использованием WPF и Entity Framework.

## 🛠 Технологии

* C#
* WPF (MVVM)
* Entity Framework (Code First)
* SQL Server

## 📂 Структура проекта

* Models — модели данных
* Views — интерфейс (XAML)
* ViewModels — логика UI
* Services — бизнес-логика
* Data — работа с базой данных

## ⚙️ Функционал

* Создание задач
* Просмотр списка задач
* Редактирование задач
* Удаление задач
* Изменение статуса
